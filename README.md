# Fraud Detection Chatbot

Начинаю изучать LangChain/LangGraph. Начну с малого и по чуть - чуть. Хочу постепенно сделать полноценный RAG-сервис
Консольный чат-бот для анализа банковских транзакций на основе LLM.
Бот принимает описание транзакции в свободной форме, анализирует её и возвращает структурированный результат с оценко**й риска.**

## Возможности

- Анализ транзакций в режиме реального времени
- Структурированный JSON ответ с оценкой риска от 1 до 10
- Память диалога — бот помнит контекст предыдущих транзакций
- Автоматическая блокировка при высоком уровне риска

## Стек

- Python 3.10+
- LangChain
- LangChain Groq
- Pydantic

## Установка

```bash
git clone https://github.com/boshka22/langchain_start
cd fraud-detection-chatbot
pip install -r requirements.txt
```

Создай файл `.env` в корне проекта:

```
GROQ_API_KEY=твой-ключ
```

Получить бесплатный ключ можно на [console.groq.com](https://console.groq.com)

## Запуск

```bash
python main.py
```

## Пример работы

```
Транзакция: 150000 рублей, Anon Casino, Кипр
Транзакция заблокирована: Транзакция в онлайн-казино за рубежом

Транзакция: а если сумма будет 5000?
Транзакция заблокирована: Транзакция в онлайн-казино за рубежом

Транзакция: 1000 рублей, Пятёрочка, Россия
Транзакция одобрена. Риск: 2/10
```

## Структура ответа

```json
{
    "risk_score": 8,
    "is_suspicious": true,
    "reason": "Транзакция в онлайн-казино за рубежом",
    "action": "block"
}
```
